# ATLAS — Frontend

Frontend Next.js del sistema **ATLAS** (Sistema agéntico de apoyo para la orientación normativa y jurisprudencial sobre playas en Colombia).

## Desarrollo

```bash
bun dev        # http://localhost:3000
bun build      # build de producción
bun lint       # eslint
```

Requiere `frontend/.env.local` con las variables de Firebase y la URL de la API. Ver variables en el README raíz del proyecto.
